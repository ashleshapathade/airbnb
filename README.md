# 🏡 Airbnb Rental Listing Web Application

## 📖 Description
A full-stack Airbnb-style web application that enables users to discover, create, and manage rental property listings. The platform provides secure authentication, dynamic listing management, and cloud-based image storage for a smooth and scalable user experience.

---

## 🚀 Live Demo
🌐 https://airbnb-xzg6.onrender.com/listings  
🚀 Deployed on Render

---

## ✨ Key Features
- 🔐 Session-based Authentication (Login/Signup)
- 🏠 Create, Update, Delete Property Listings
- 🖼️ Image Upload using Multer & Cloudinary
- 🔍 Browse & Explore Listings
- ⭐ Add, Edit & Delete Reviews
- 👤 User-specific Listing Management
- 📱 Fully Responsive UI

---
